Prerequisites

The following services are needed by PopIt and you’ll need to install them manually. If you use the instructions for a specific platform then you’ll be guided through the steps needed.

Server and DB

  • Node.js - the platform that the codebase runs on.
  • MongoDB - the database we store all the data in.

Dev tools

If you want to contribute code or run the test suite you’ll need the following additional deps:

  • Ruby used for the browser based tests
  • Watir WebDriver - to run the browser tests
  • Chrome - browser used for the automated tests
  • Chrome Driver - interface for the webdriver code
  • Compass - to compile the stylesheets

Documentation

The documentation part of the site is constructed using:

  • Jekyll which turns compiles the documentation source into the doc site
  • kramdown for converting to HTML, and the pretty syntax highlighting
