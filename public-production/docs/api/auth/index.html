API Auth

For write operations on the API you need to be authenticated. This can either be done using Basic Authentication or by cookie auth. Basic is probably best for scripts and programatic access, cookie based for Ajax calls etc.

We will probably need to add token based auth (#113) in the future, as well as adding CSRF protection (#112). Tickets have been created for these in the issue tracker.
